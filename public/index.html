Ice cream API

This is an API about ice cream and it's nutritional content

Endpoints: 

/allIceCreams                                           // Shows all ice creams and the data
/allIceCreams/icecream:id                     // Shows a specific ice cream and it's data. Example: /allIceCreams/icecream1
/allIceCreams/add                                  // Shows a form that will let you add new ice creams to the API
/allIceCreams/delete/:id                        // Deletes ice cream with the chosen id. Example: /allIceCreams/delete/1
